Detailed Description

template<typename T, typename Allocator = std::allocator<T>>
class edsp::filter::moving_average< T, Allocator >

This class implement a cumulative moving average (rolling average or running average) filter.

Given a series of numbers and a fixed subset size, the first element of the moving average is obtained by taking the average of the initial fixed subset of the number series. Then the subset is modified by "shifting forward"; that is, excluding the first number of the series and including the next value in the subset.

The filter implement a basic cumulative unweighted mean of the previous N data, where N is the length of the moving window:

\[ \lambda_i = \frac{x_{n} + x_{n-1}+ \cdots + x_{i-(N-2)} + x_{i-(N-1)}}{N} \]
